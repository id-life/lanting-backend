Test Instructions

  1. Make sure your backend is running on http://localhost:8000
  2. Ensure your GitHub OAuth app is configured with callback URL: http://localhost:8000/api/auth/github/callback
  3. Click "Login with GitHub" to test the OAuth flow
  4. After successful login, you'll be redirected back with a token
  5. The token will be displayed and stored in localStorage
  6. You can test authenticated API calls using the token
